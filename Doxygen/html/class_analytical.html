Detailed Description

It's a derived class from Solve /nIt's purpose is to calculate the solution of the problem with the analytical solution
-T is the solution of the problem
-m is the number that will remplace infinity in the sum of the analytical solution

The Analytical class provide
-a basic constructor based on the ine define in the solver class
-a solve function that will calculate an approximation of T base on the analytical solution, T will be more and more accurate as m grows
