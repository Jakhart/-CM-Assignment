Detailed Description

It's a class designed to solved the problem with the Crank-Nicholson method It's derived from the Implicit class The Crank-Nicholson class provides:
-basic constructor based on the Solve class constructor
-a virtual solve function that solve the problem with the Crank-Nicholson scheme
