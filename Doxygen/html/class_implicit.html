Detailed Description

This class is derived from the Solve class and is the base class of the two implicit scheme Class, Crank-Nicholson and Laasonen.

The Implicit class provides:
-basic constructor derived from Solve
-a virtual function solve() that will be use in the derived function to solve the problem
-a

See also
Diagonalization(Vector &T) function that will be used in both derived class to calculate the first part of the thomas Algorithm (Diagonalization and resolving the equation Ly = b )
