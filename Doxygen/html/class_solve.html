Detailed Description

It's the main base classes of the program
It implements the data needed to solve the problem with each method
In those date we can find:
-D the diffusity in ft^2/hr
-Tin The initial temperature
-Tsun The temperature of of the two surfaces of the wall at x0 and xn
-dt The gap in time between n and n+1
-dx The gap in space between i and i+1

The Solve class provides:
-basic constructor that will be reused in all derivated class
-set and get function for each variable set by the user
-a virtual function solve() that will be use in the derived function to solve the problem
